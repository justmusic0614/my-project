# Market Digest

股票追蹤與分析系統，整合財報、籌碼面數據與智慧分析。

## 快速開始

### Telegram 指令
```
/financial        # 日報
/weekly          # 週報
/analyze 2330    # 單檔分析
/alerts          # 異常檢查
```

### 建立追蹤清單
```
/watchlist add 2330 2454 2408
```
