# Market Digest 使用手冊

**版本：** Phase 2  
**日期：** 2026-02-04

## 快速開始

**建立清單 → 查看日報 → 檢查異常**

```
/watchlist add 2330 2454
/financial
/alerts
```

## Telegram 指令

### Watchlist 管理
```
/watchlist list              # 列出清單
/watchlist add 2330 2454     # 新增
/watchlist remove 2330       # 移除
```

### 報告查詢

**/financial** - 日報
- 財報 + 籌碼 + 分析
- 籌碼概況統計
- 異常提醒
- 個股詳細報告

**/weekly [days]** - 週報（預設 5 日）
- 週度法人動向
- 連續買超/賣超
- 整體概況

**/analyze <代號>** - 單檔分析
- 綜合評分（0-100）
- 融資分析
- 法人分析
- 異常提醒

### 智慧提醒

**/alerts** - 檢查異常
- 8 種檢測規則
- 24 小時冷卻

**檢測項目：**
- 🔴 融資使用率 > 30%
- 🔴 外資買賣超 > 10000 張
- 🔴 籌碼評分 < 30
- 🟡 融資大增 > 10%
- 🟡 投信買超 > 2000 張
- 🟡 融券回補 > 15%

### 早報搜尋

**/query <關鍵字>** - 搜尋早報
```
/query 聯發科                # 近 7 天
/query 台積電 --days 30      # 近 30 天
```

## 日常流程

**早上（09:00）：**
```
/financial        # 查看日報
/alerts          # 檢查異常
```

**下午（16:00）：**
- 系統自動檢測異常
- 發現問題時推播

**週末：**
```
/weekly          # 週報回顧
```

**隨時：**
```
/analyze 2330    # 單檔分析
/query 聯發科    # 搜尋早報
```

## 評分與異常解讀

### 評分系統
- **🟢 65+：** 偏多，建議關注
- **➖ 35-64：** 中性，持續追蹤
- **🔴 <35：** 偏空，建議觀望

**評分因素：**
融資使用率 + 法人情緒 + 異常事件

### 異常提醒意義

**高優先級（立即關注）：**
1. 融資使用率 > 30% → 投機過熱
2. 外資買賣超 > 10000 張 → 機構大動作
3. 籌碼評分 < 30 → 極度不健康

**中優先級（留意觀察）：**
4. 融資使用率 > 20% → 偏高
5. 融資大增 > 10% → 投機買盤
6. 投信買超 > 2000 張 → 機構看好
7. 融券回補 > 15% → 空頭回補（利多）

### 週報解讀
- 連續買超 ≥ 3 日 → 法人持續看好
- 週度累計 > 10000 張 → 重大動向

## 常見問題

**Q: 股票沒有數據？**
確認代號正確、等待數分鐘重試

**Q: 冷卻機制？**
同股票同異常 24 小時內不重複通知

**Q: 評分計算？**
基準 50 + 融資(-20~+10) + 法人(-15~+15)

**Q: 週報顯示「近 1 日」？**
快取不足，3-5 天後完整

**Q: 調整閾值？**
修改 `alert-monitor.js` 的 `ALERT_RULES`

**Q: 新增指標？**
需確認資料來源、修改程式、測試驗證
